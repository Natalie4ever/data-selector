# 数据加工助手 - 测试用例文档

## 1. 概述

### 1.1 测试范围
本文档覆盖数据加工助手系统的所有功能模块，包括：
- 用户认证与权限管理
- SQL 查询管理（增删改查）
- 二级菜单管理
- 查询执行与结果展示
- 时间筛选与日期分组
- 数据导出
- 审计日志

### 1.2 测试环境准备
```
默认测试账号：
- EHR 号：1234567
- 密码：Password01!

管理员账号（审计白名单用户）：
- EHR 号：1234567
- EHR 号：1234568
```

### 1.3 测试工具
- API 测试工具（如 Postman、curl）
- 浏览器测试（Chrome/Edge DevTools）
- 后端服务器：`cd backend && python -m uvicorn main:app --reload`
- 前端服务器：`cd frontend && npm run dev`
